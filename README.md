# Calculator

Современный веб-калькулятор с элегантным дизайном и интуитивным интерфейсом.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## Возможности

- **Базовые операции** — сложение, вычитание, умножение, деление
- **Проценты** — быстрый расчёт процентов
- **Тёмная тема** — стильный дизайн, приятный для глаз
- **Клавиатура** — полная поддержка ввода с клавиатуры
- **Адаптивность** — корректное отображение на любых устройствах

## Управление

| Клавиша | Действие |
|---------|----------|
| `0-9` | Ввод цифр |
| `+ - * /` | Операции |
| `Enter` или `=` | Вычислить |
| `Backspace` | Удалить символ |
| `Escape` | Очистить |
| `%` | Процент |
| `.` | Десятичная точка |

## Запуск

Откройте файл `index.html` в любом современном браузере.
